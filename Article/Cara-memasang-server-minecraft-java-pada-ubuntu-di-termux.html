Cara Pasang Server Minecraft Java di Smartphone Menggunakan Ubuntu di Termux

Membuat sebuah smartphone bisa menjadi server untuk minecraft java, Berikut ini cara caranya.

Harus diperhatikan.

Tidak semua smartphone mampu untuk melakukan ini !

  • Smartphone harus memiliki setidaknya 6GB RAM.
  • Memiliki ROM atau memori internal 128GB.
  • Minimum kecepatan chipset 2.2 Ghz ( Untuk kurang dari 6 player yang ada didalam server ).

Apa yang terjadi jika tidak memenuhi spesifikasi minimum diatas ?

Server akan tetap bisa berjalan, Tetapi server tidak bisa berjalan mulus dan dipastikan akan ada sedikit keterlambatan tick didalam server

1. Download termux terlebih dahulu di Play Store

Setelah mendownload termux, Masuk ke aplikasinya dan ketikkan perintah dibawah ini

pkg update

pkg upgrade

termux-setup-storage

Penjelasan :

Perintah pkg update berfungsi untuk mengupdate paket, Hal ini bertujuan agar tidak terjadi error atau kesalahan pada saat menginstall sesuatu.

Perintah pkg upgrade berfungsi untuk mengupgrade paket dan sama tujuannya seperti perintah pertama.

Perintah termux-setup-storage berfungsi untuk memberikan akses ke storage atau penyimpanan kepada termux, Hal ini akan mempermudah proses pemindahan file yang sudah didownload langsung melalui smartphone ke dalam termux.

2. Install ubuntu untuk termux

Setelah semua perintah diatas sudah dijalankan, Ketikkan perintah dibawah ini

pkg install wget curl proot tar -y

wget https://raw.githubusercontent.com/tuanpham-dev/termux-ubuntu/master/ubuntu.sh

chmod +x ubuntu.sh

bash ubuntu.sh -x xfce4

Penjelasan :

Perintah pkg install wget curl proot tar -y berfungsi untuk menginstall paket wget, curl, proot, dan tar

pkg install berfungsi sebagai perintah untuk menginstall paket.

wget berfungsi sebagai alat pengunduh file dari web.

curl berfungsi sebagai alat pentransfer data dengan url.

proot berfungsi sebagai alat untuk menjalankan program dengan lingkungan root palsu.

tar berfungsi sebagai alat untuk membuat dan mengekstrak file atau arsip bertipe tar.

-y berfungsi mengkonfirmasi instalasi tanpa meminta persetujuan.

Perintah

wget https://raw.githubusercontent.com/tuanpham-dev/termux-ubuntu/master/ubuntu.sh
berfungsi untuk menginstall file bernama ubuntu.sh

wget berfungsi untuk mengunduh file dari web.

https://raw.githubusercontent.com/tuanpham-dev/termux-ubuntu/master/ubuntu.sh
url file skrip ubuntu.sh yang diunduh.

Perintah chmod +x ubuntu.sh berfungsi untuk mengubah izin file menjadi eksekutabel, sehingga dapat dijalankan.

chmod berfungsi untuk mengubah izin atas file agar bisa dibaca dan dialankan di sistem operasi berbasis Unix/Linux.

+x sebuah argumen yang berfungsi untuk perintah chmod yang berarti tambah izin eksekusi.

Perintah bash ubuntu.sh -x xfce4 berfungsi untuk menjalankan file bernama ubuntu.sh dengan konfigurasi.

bash Sama seperti ./nama-file. dan berfungsi untuk menjalankan sebuah file.

ubuntu.sh nama file.

-x xfce4 berfungsi untuk mengkonfigurasi lingkungan dekstop xfce4 yang akan diinstall.

1. Kenapa memilih versi ubuntu ini ?

2. kenapa harus menginstall ubuntu terlebih dahulu ?

Jawaban :

1. Saya memilih versi ubuntu ini karena menurut saya ini versi ubuntu untuk termux yang paling lengkap.

2. Karena ini untuk memudahkan dan kenyamanan pada saat menjalankan server, Dan agar tidak terjadi error atau masalah pada saat penginstallan server atau penginstallan hal lainnya yang diperlukan untuk menjalankan server.

Tunggu proses penginstallan ( ini memang cukup lama ), Setelah itu konfigurasi ubuntu sesukamu.

3. Install java dan server minecraft java

Setelah selesai menginstall ubuntu di termux, Pastikan kamu sudah menjalankan ubuntu.

ketikan beberapa perintah ini di dalam terminal ubuntu.

1. Update sistem ubuntu

sudo apt update && sudo apt upgrade -y

2. Install java

sudo apt install openjdk-8-jdk

Periksa versi java

java -version

Jika java sudah diinstall dan terdeteksi oleh ubuntu, Lanjut ke langkah selanjutnya.

3. Install server

mkdir server-mc-java

cd server-mc-java

wget https://launcher.mojang.com/v1/objects/1b557e7b033b583cd9f66746b7a9ab1ec1673ced/server.jar

4. Jalankan server

java -Xmx4024M -Xms4024M -jar server.jar nogui

Pada saat ini kamu akan melihat pesan di terminal ubuntu untuk menyetujui file eula.txt

Ketikkan perintah ini untuk menyetujui eula

nano eula.txt

Kamu akan melihat script ini didalam file eula.txt

eula=false

Ubah isi eula menjadi true seperti ini

eula=true

Jalankan kembali server

java -Xmx4024M -Xms4024M -jar server.jar nogui

Kamu juga bisa mengkonfigurasi server dengan cara mengakses file bernama server.properties, Ketikkan perintah ini

nano server.properties

Cari baris script ini didalam file server.propeties

gamemode=survival

difficulty=easy

pvp=true

max-player=20

view-distance=10

online-mode=true

motd=A Minecraft Server

Penjelasan :

Untuk gamemode=survival bisa diubah ke spectator, creative, hardcore ,dan survival.

Untuk difficulty=easy bisa diubah ke normal, hard, dan easy.

Untuk pvp=true bisa diubah ke false atau true, false akan membuat player tidak bisa menyerang player lain, sedangkan true akan membuat player bisa menyerang player lainnya.

Untuk max-player=20 bisa diubah sesukamu dengan isi sebagai angka, Jika kamu membuat isinya 10 berarti hanya 10 player yang bisa masuk ke server kamu.

Untuk view-distance=10 bisa diubah sesukamu dengan isi sebagai angka, Sebenarnya ini sama dengan pengaturan chunk di gamenya, Jika kamu mengisi kurang dari 5 maka tampilan dunianya akan menjadi lebih kecil dan sebaliknya jika kamu mengisinya lebih dari 8 maka tampilan dunia akan menjadi lebar.

Untuk online-mode=true bisa diubah ke false atau true, Jika kamu mengubahnya ke false maka player dengan minecraft java bajakan akan bisa masuk.

Untuk motd=A Minecraft Server bisa diubah sesukamu dengan isi sebagai huruf, Ini adalah tampilan nama server kamu.

Cara masuk ke server milikmu

  • Masuk ke pengaturan smartphone
  • Cari tentang ponsel
  • Masuk ke informasi status
  • Kamu akan melihat alamat IP, Salin alamat ip tersebut dan masuk ke multiplayer di minecraft
  • Lalu klik add server, Pada server name biarkan saja begitu dan pada server address isikan alamat IP yang kamu sudah salin tadi
